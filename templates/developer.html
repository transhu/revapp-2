Complete guide to CritiqueSpark's features and architecture

About CritiqueSpark

CritiqueSpark is a modern web platform designed for developers to share and discuss projects, receive feedback, and build a collaborative community. Built with Flask and modern web technologies, it offers a rich set of features for both users and developers.

Core Features

User Authentication

  • Secure login and registration
  • Session management
  • Password protection
  • User profile management

Post Management

  • Create and edit posts
  • Rich text formatting
  • Project link sharing
  • Post visibility controls

Interactive Comments

  • Threaded comments
  • Reply functionality
  • Like/Unlike comments
  • Real-time updates

Voting System

  • Upvote/Downvote posts
  • Vote tracking
  • User vote history
  • Trending posts

Profile System

  • Custom avatars
  • Bio and contact info
  • Activity tracking
  • Profile customization

Chat System

  • Real-time messaging
  • User-to-user chat
  • Message history
  • Online status

Technical Stack

Backend

  • Python 3.x
  • Flask Framework
  • JSON Data Storage
  • Session Management

Frontend

  • HTML5/CSS3
  • JavaScript (ES6+)
  • Bootstrap 5
  • Custom Components

API Reference

Authentication

POST /login
POST /signup
POST /logout

Posts

GET /posts
POST /create_post
GET /post/{post_id}
POST /upvote/{post_id}
POST /downvote/{post_id}

Comments

POST /comment/{post_id}
POST /comment/{post_id}/{comment_id}/reply
POST /comment/{post_id}/{comment_id}/like

User Profile

GET /mypage
POST /save-profile
GET /settings
POST /save_settings

Chat System

GET /chat
GET /chat/messages/{user}
POST /chat/send/{user}
